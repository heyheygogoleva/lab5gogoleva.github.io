Detailed Description

Класс Builder, который создает объект и добавляет соответствующие продукты пицце. Вместо замысловатого конструктора мы можем создать экземпляр объекта и добавить необходимые компоненты с помощью функций.

Мы вызываем конструкцию каждого модуля отдельно, после создания объекта.  Класс Pizza определен с некоторыми значениями по умолчанию
